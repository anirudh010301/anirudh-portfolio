research.md — SELECT * FROM publications WHERE type = 'survey'
AI-Augmented Data Engineering
How LLMs, MLOps, and Intelligent Agents Are Reshaping Modern Pipelines
● published 2024 survey paper data engineering · AI / LLMs
LLM Code Generation AI Data Quality Schema Inference Agentic Self-Healing MLOps Observability
The data engineering discipline is undergoing a structural transition from deterministic programming toward probabilistic supervision. Where engineers once wrote explicit transformation logic, static quality rules, and reactive monitoring alerts, they increasingly collaborate with AI systems that infer schemas from sample data, detect anomalies without pre-specified rules, generate pipeline code from natural language, and — at the frontier — autonomously diagnose and repair failures.

This survey characterises this transition across four capability domains: pipeline automation and LLM code generation, AI-powered data quality and observability, schema inference and metadata intelligence, and agentic self-healing architectures. Our central thesis is that the appropriate model for AI integration in data engineering is not autonomous execution but human-AI collaborative supervision — a paradigm with significant implications for how teams are composed, how pipelines are governed, and how the profession is defined.
